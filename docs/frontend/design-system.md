# Design System

## Visual Direction
- Glass UI cards (`backdrop-blur`, transparent layers, soft shadows)
- Blue accent (`#4A90E2` equivalent HSL)
- Light/dark theme variables

## Typography
- Heading: `Noto Sans Mono`
- Body: `IBM Plex Serif`
- Heading sizes: 32-40 px on hero, dashboard titles 22-24 px
- Body sizes: 14-16 px

## Motion
- `ease-out`
- 200-300ms transitions
- slide/blur/word reveal animations applied once

## Color Tokens
- Background, foreground, border, card
- Status tokens: success, warning, destructive

## Dark Mode
- Toggle in header
- Persisted via Zustand + local storage
