# Pages and Flows

## Public
- `/`: landing page (project info, goal, contact)
- `/login`: authentication page

## Protected
- `/dashboard`: stats + AI assistant tabs
- `/dashboard/dissertations`: advanced search + registry table

## User Journeys
1. User logs in.
2. Dashboard shows global stats and AI chat.
3. User opens dissertations page.
4. Applies multi-filter parameters.
5. Reviews table rows and details modal.

## Search UX
Search form covers fields:
- title
- problem
- proposal
- annotation
- conclusion
- keywords
- author
- supervisor
- university
- scientific direction

Plus advanced parameters:
- registry category
- expert rating slider
- region selector
- visibility toggle
